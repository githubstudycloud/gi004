## Spring Boot 4.0 核心特性

- **完全模块化**: 70+ 个独立模块，告别臃肿的单体 JAR
- **Spring Framework 7.0**: 全新基础框架
- **Java 25 支持**: 向前兼容 Java 17
- **JSpecify 空安全**: 全面的空值检查
- **API 版本控制**: 内置 API 版本管理
- **Kotlin 2.2 基线**: 一流的 Kotlin 支持
